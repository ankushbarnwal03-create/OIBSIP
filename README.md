# OIBSIP Level 1 Tasks
This repository contains all tasks completed for Level 1 of the OIBSIP project.

## Task 1: Landing Page
- Description: Created a landing page using HTML and CSS.
- Features: Header, Hero section, Features section, Footer.

## Task 2: Personal Portfolio
- Description: Developed a personal portfolio showcasing skills, projects, and contact information.
- Features: Image, About section, Skills, Projects, Contact info.

## Task 3: Temperature Converter
- Description: Built a temperature converter using HTML, CSS, and JavaScript.
- Features: Input field, unit selection, conversion logic, Kelvin option.
